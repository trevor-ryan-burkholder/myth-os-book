A [Signal](/wiki/signal/) that has appeared enough times across different situations, with consistent enough character, to read as the same source activating.

Consistency is the marker. Not clarity. Not detail. Not meaning. Same conditions, same tone, same speed of arrival, same overall quality. That tells you the same source is firing each time. The source does not need to be named or understood. It needs to be recognized as the same thing showing up again.

Repeated signals are the primary unit of recognition in the practice. "Pattern" in this book refers to repeated signals over time. Not anything in [The space](/wiki/the-space/) itself.

The [Somatic channel](/wiki/somatic-channel/) cross-checks. Repeated signals usually carry a consistent physical signature, a specific tightening, a location where the response lands, a breathing change particular to that signal. Cognitive recognition plus somatic recognition gives two independent readings. When they agree, the call is reliable.

Naming short-circuits this. Once something has a label, new observations get filtered through it. You stop noticing and start confirming. The label feels like clarity. It replaces accuracy with a fixed idea that no longer updates. The character of the signal is the data. The name is a conclusion you have not earned.
