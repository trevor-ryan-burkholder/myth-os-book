# Repeated signal

A [[signal]] that has appeared enough times across different situations, with consistent enough character, to be recognized as the same source activating.

Consistency is the marker. Not clarity, not detail, not meaning. The same signal appearing under similar conditions, with the same approximate tone, the same speed at which it arrives, the same overall quality of the experience, is telling you that the same source is being activated each time. That source does not need to be named or understood to be real. It only needs to be recognized as the same thing showing up again.

Repeated signals are the primary unit of recognition in the practice. The word "pattern" in this book refers specifically to repeated signals over time, not to anything in [[the-space]] itself.

The [[somatic-channel]] is useful here. Repeated signals often carry a consistent physical signature, a specific quality of tightening, a location in the body where the response lands, a change in breathing that is particular to that signal rather than generic. Noticing the body's response alongside the cognitive recognition gives two independent readings of whether something is the same or different. When both agree, the identification is more reliable.

Naming short-circuits this. Once something has a label, even a provisional one, new observations get filtered through it. Instead of noticing what is actually happening, you start confirming what you already decided. The label feels like clarity, but it replaces accuracy with a fixed idea that stops updating. The character of a signal is the data. The name is a conclusion you have not yet earned.
