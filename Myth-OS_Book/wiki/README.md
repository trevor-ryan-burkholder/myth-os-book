# Myth-OS Wiki

A reference for the system described in the Myth-OS manuscript. Each page is short by design. Concepts link to related concepts. Topic pages synthesize material that crosses chapters.

This is not a summary of the book. It is a way to navigate what the book describes when you need to find one thing quickly, or when you want to see how two things connect.

## How to read this wiki

- Concept pages define one thing, link to related things, and point to the chapter where the concept is introduced.
- Topic pages synthesize across chapters. Use them when the question is broader than one term.
- The [[chapters-index]] is the linear path through the book. Use it when you want sequence rather than reference.

Nothing here replaces reading the manuscript. The wiki is a map. The territory is in the chapters.
